# SA COD Converter

**SA COD Converter** is a modern, responsive web tool that allows the user to enter a checkout date and instantly calculates the number of days and hours remaining until a guest’s checkout.

## Features
- User enters checkout date to calculate remaining time for a guest
- Displays **Days** and **Hours** separately
- Validates past dates
- Instant auto-calculation
- Created by CTC_Klay
